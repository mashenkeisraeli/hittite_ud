# hittite_ud
This is to represent a project dedicated to the development of a Universal Dependencies (UD) treebank for Hittite, the most ancient of Indo-European languages attested in writing. To date, only a single Hittite treebank utilizing UD annotation exists, and it is built on examples from a textbook (Andersen, Rozonoyer 2020), not on critical editions of the Hittite texts. Previously, a pilot project for UD annotation of Hittite was initiated by a co-author of this paper within the PROIEL databases framework (Inglese et al. 2018). 
This UD treebank is being built on the material of Hittite letters, as published by (Hoffner 2009) and represented in the Annotated Corpus of Hittite Clauses (ACHC, http://hittitecorpus.com). It includes updates for the latest UD annotation format (UD v. 2.9).
