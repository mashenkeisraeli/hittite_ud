# hittite_ud
This is to represent a project dedicated to the development of a Universal Dependencies (UD) treebank for Hittite, the most ancient of Indo-European languages attested in writing. To date, only a single Hittite treebank utilizing UD annotation exists, and it is built on examples from a textbook (Andersen, Rozonoyer 2020). A pilot project for UD annotation of Hittite was initiated by a co-author of this paper within the PROIEL databases framework (Inglese et al. 2018). However, a UD treebank on the material of critical editions of original Hittite texts is still a desideratum. This paper presents solutions developed for UD features specific to Hittite, addressing issues highlighted in (Inglese et al. 2018), and includes updates for the latest UD annotation format (UD v. 2.9).
